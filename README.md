## What it does

Video Preview is built for the moment you have a clip and need to do something with it — fast.

**Scrub frame by frame.** Use `,` and `.` to step one frame at a time. Arrow keys seek by seconds.

**Trim without re-encoding.** Set in and out points, export the selection. Original quality preserved.

**Crop with precision.** Draw a region, lock to 16:9, 4:3, or 1:1, or go freeform. Export immediately.

**Optimize for sharing.** Right-click any video in Explorer — remux to MP4, strip metadata, remove audio, compress for web. Each operation creates a new file. The original is never touched.

**Screenshot any frame.** `Ctrl+Shift+S` saves the current frame as a PNG.

**Keyboard-first.** Every action has a shortcut. Press `?` to open the command palette. Hold a modifier key to see which shortcuts apply.

## Why not VLC / Premiere / [other tool]?

Those tools are great at what they do. Video Preview is for the gap between "I need to watch this clip" and "I need to quickly process it before moving on." It opens fast, gets out of the way, and does the four or five things you actually need — without a project file, a timeline, or an account.
